## Reportar una Vulnerabilidad

Si encuentras un problema de seguridad, **no abras un Issue público**. Esto podría exponer el sistema antes de que podamos corregirlo.

Para reportar vulnerabilidades de forma responsable:

1. Envía un correo electrónico a **felipe28769@gmail.com** con el asunto **"REPORTE SEGURIDAD - HORSETRUST"**.
2. Incluye una descripción detallada del hallazgo y los pasos necesarios para reproducirlo.
3. El equipo de DevOps revisará el reporte y se pondrá en contacto contigo en un plazo estimado de 48-72 horas.

Agradecemos tu colaboración para mantener este proyecto seguro durante la fase de simulación.
